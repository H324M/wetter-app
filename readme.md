# Wetter-App nach PLZ

Kleine Webanwendung zur Anzeige einer Wettervorhersage für eine deutsche Postleitzahl.

## Funktionen
- Wetter für heute
- Wetter für morgen
- 7 weitere Tage
- Temperaturdiagramm
- Speicherung der letzten PLZ im Browser

## Nutzung
Einfach die GitHub-Pages-URL im Browser öffnen.

## Technologien
- HTML / CSS / JavaScript
- Open-Meteo API
- Zippopotam API
